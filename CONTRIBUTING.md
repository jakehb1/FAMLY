# Contributing to Famly

Thank you for your interest in contributing to Famly!

## Development Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables (see `.env.example`)
4. Start development server: `npm start`

## Code Style

- Use TypeScript for all new code
- Follow ESLint rules (run `npm run lint`)
- Format code with Prettier (run `npm run format`)
- Use functional components with hooks
- Follow the existing code structure

## Commit Messages

- Use clear, descriptive commit messages
- Prefix with type: `feat:`, `fix:`, `docs:`, `refactor:`, etc.

## Testing

- Test on iOS simulator before submitting
- Test on web before deploying
- Verify all user flows work correctly

## Pull Requests

1. Create a feature branch
2. Make your changes
3. Run `npm run lint` and `npm run type-check`
4. Submit PR with clear description
